====================================================
        TripCrafter - AI 여행 플래너 시작하기
====================================================

안녕하세요! TripCrafter를 시작하는 방법입니다.

--------------------------------------------------
📋 체크리스트
--------------------------------------------------
□ Windows 10/11 사용 중
□ WSL2 활성화됨 (이미 이 파일을 보고 계시면 OK!)
□ Docker Desktop 설치 필요

--------------------------------------------------
🔧 Docker Desktop 설치 (아직 안하셨다면)
--------------------------------------------------
1. 웹브라우저에서 접속:
   https://www.docker.com/products/docker-desktop/

2. "Download for Windows" 클릭

3. 다운로드된 파일 실행하여 설치

4. 설치 완료 후 Docker Desktop 실행
   (Windows 시작 메뉴에서 찾을 수 있습니다)

5. 시스템 트레이에 고래 아이콘이 나타나면 준비 완료!

--------------------------------------------------
🚀 TripCrafter 실행
--------------------------------------------------
Docker Desktop이 실행 중인 상태에서:

1. 이 터미널에서 다음 명령어 입력:
   ./start-app.sh

2. 처음 실행 시 10-15분 정도 걸립니다
   (커피 한잔 하고 오세요! ☕)

3. "✅ TripCrafter가 성공적으로 시작되었습니다!"
   메시지가 나타나면 완료!

4. 웹브라우저에서 접속:
   http://localhost:3000

--------------------------------------------------
💡 사용 팁
--------------------------------------------------
• 여행 계획 예시: "도쿄 4박5일 신혼여행 계획해줘"
• 서버 중지: docker-compose down
• 다시 시작: ./start-app.sh
